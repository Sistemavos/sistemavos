# SistemaVOS

Sistema de Gestion Integral (CRM/ERP) para PyMEs de Latinoamerica.

## Estructura

```
index.html                        → Landing page / sitio marketing
app.html                          → Aplicacion completa (SPA)
vercel.json                       → Configuracion de deploy en Vercel
guia-bienvenida-sistemavos.pdf    → Guia de bienvenida para usuarios
```

## Deploy

Este proyecto esta configurado para deploy automatico en **Vercel**:

1. Subi este repositorio a GitHub
2. Conecta el repo desde [vercel.com](https://vercel.com)
3. Hace clic en "Deploy" — listo
